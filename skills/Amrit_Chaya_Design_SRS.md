# AMRIT CHAYA — WEBSITE DESIGN & SRS GUIDE

# Brand Understanding

## Logo Analysis
The **Amrit Chaya** logo represents:
- Spiritual shelter
- Peace & growth
- Divine protection
- Nature & seva
- Calmness and trust

The tree identity creates a feeling of:
- आश्रय (Shelter)
- सेवा (Service)
- अध्यात्म (Spirituality)
- प्रकृति (Nature)
- जीवन (Life)

So the website should feel:
- Divine
- Premium
- Peaceful
- Spiritual
- Trustworthy
- Modern but traditional

---

# Recommended Design Style

## Design Theme

### Core Design Language
Use:
- Minimal spiritual aesthetics
- Large peaceful imagery
- Temple-inspired sections
- Smooth transitions
- Gold + Green + Saffron palette

Avoid:
- Crowded UI
- Too many animations
- Random colors
- Corporate tech feel
